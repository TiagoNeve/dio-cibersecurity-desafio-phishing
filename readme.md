# Desafio de Phishing usando SETOOLKIT

## Objetivo
Clonar o gmail e usar isso como base para a captura do acesso das vítimas

## Comandos
sudo su -> Se tornar usuário root no terminal, essencial para conseguir executar comandos sem impeditivo

setoolkit -> Iniciar o aplicativo CLI para executar ferramentas de Phishing

Passo a passo no menu: 1 -> 2 -> 3 -> 1 -> Colocar o IP para conexão -> Escolher o template do google.

## Processo
Colocar o IP em algum registro de domínio específico
Enviar um e-mail com o link específico
Quando a pessoa clicar, irá cair no site fake.
Receberemos essa senha do usuário

## Como evitar
Olhe sempre os links antes de colocar qualquer senha.
Habilite o recurso de MFA, pois mesmo que perca a senha, o invasor precisará passar pelo MFA após tentar acessar com o usuário e senha.
